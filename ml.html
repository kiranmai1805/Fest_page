Machine Learning Hackathon @ INVENTUM 2K24

Departments of CSE (AI & ML, DS), ANITS is proud to announce a Machine Learning Hackathon for students as a part of college technical festival INVENTUM 2K24.

This event provides a platform for participants to showcase their expertise in machine learning and data science. Evaluation will be based on participants' ability to innovate and build machine learning models to solve real-world problems effectively.

Participants are allowed to use the programming language PYTHON and must have a Kaggle account.

Workshops: 23/02/2024 and 24/02/2024

First Round: Coding Challenge (3 hrs) – 27/02/2024

Second Round: ML Hackathon – 29/02/2024

Registration Fee: Rs. 60/- per head

Prizes: 1st - ₹4000, 2nd - ₹3000
